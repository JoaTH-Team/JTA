class Writer

package jta.objects.dialogue

extends TextTyper

@:directlyUsed

Represents a dialogue writer that displays text sequentially, allowing the player to advance or skip through dialogue pages. Handles the display of dialogue using different typers and invokes a callback when the dialogue is finished.
