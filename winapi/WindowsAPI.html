@:value({ b : 25, g : 25, r : 25 })staticsetWindowTransparent(r:Int = 25, g:Int = 25, b:Int = 25):Void

Makes a specific color in the window fully transparent using color keying.

Any pixel matching the given RGB color will be rendered as transparent at the OS level, similar to a chroma key effect.

The default color is RGB(25, 25, 25), a near-black that is unlikely to appear in normal UI content.

Parameters:

r

The red component of the key color (0-255). Default: 25.

g

The green component of the key color (0-255). Default: 25.

b

The blue component of the key color (0-255). Default: 25.
