@:value({ setBorder : true, setHeader : true })staticinlinesetWindowBorderColor(color:Array<Int>, setHeader:Bool = true, setBorder:Bool = true):Void

Sets the header and/or border to a color of your choosing. (Only Windows 11 supports this).

Parameters:

color

The color of the window border/header. organized as [R (0 to 255), G (0 to 255), B (0 to 255)].

setHeader

Do you want to set the header to the specified color?

setBorder

Do you want to set the border to the specified color?

Parameters:

cornerType

The corner type to use.
