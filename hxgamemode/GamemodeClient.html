class GamemodeClient

package hxgamemode

@:directlyUsed@:buildXml("<include name=\"${haxelib:hxgamemode}/project/Build.xml\" />")@:include("gamemode_client.h")@:unreflective

This class provides access to the GameMode daemon, enabling processes to request and control performance optimizations on Linux systems.

Static methods

@:native("gamemode_error_string")staticerror_string():ConstCharStar

Retrieves a string describing the last error encountered by any of the GameMode functions.

Returns:

A string representing the last error message.

@:native("gamemode_query_status")staticquery_status():Int

Queries the current status of GameMode for the current process.

Returns:

0 if GameMode is inactive. 1 if GameMode is active. 2 if GameMode is active and this client is registered. -1 if the query failed.

@:native("gamemode_query_status_for")staticquery_status_for(pid:PID_T):Int

Parameters:

The process ID of the target process.

Returns:

0 if GameMode is inactive. 1 if GameMode is active. 2 if GameMode is active and this client is registered. -1 if the query failed.
