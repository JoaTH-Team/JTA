class ZipParser

package polymod.util.zip

@:directlyUsed

Parses the Central Directory Headers of a ZIP file on the file system. This allows individual files to be directly accessed without having to decompress the entire ZIP file.

Only compatible with sys targets with native file system access.

Constructor

new(fileName:String)

Variables

centralDirectoryRecords:StringMap<CentralDirectoryFileHeader>

The central directory records, as parsed from the central directory. These contain metadata about each file in the archive.

endOfCentralDirectoryRecord:EndOfCentralDirectoryRecord

The end-of-central-directory record, as parsed from the end of the ZIP file.

fileName:String

The file system path to the ZIP file.

Methods

getLocalFileHeaderOf(localFileName:String):LocalFileHeader

Read the centeral directory header for a specific file, and generate a LocalFileHeader.

Parameters:

localFileName

A filename relative to the root of the ZIP file.

Returns:

A LocalFileHeader for the specified file, or null if the file was not found.
